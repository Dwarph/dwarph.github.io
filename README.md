I'm no webdev, so forgive the hacked-together nature of this site.

## Repository layout

- **`assets/branding/`** — Favicons and PWA manifest icons.
- **`audio/`** — Soundboard clips (grouped by category).
- **`data/`** — JSON for the homepage, projects, and case study content.
- **`docs/`** — Design notes and internal scratch files (e.g. `todo.txt`).
- **`files/`** — Static downloads (e.g. CV PDF).
- **`icons/`** — Inline SVGs for footer and social links.
- **`images/`** — Site imagery, including `casestudies/` and `projects/`.
- **`js/`** — Client-side scripts for each page and shared UI (header, footer, theme).
- **`styling/`** — CSS.
- **`twitter/`** — Static Twitter archive (tweetback); URLs are stable under `/twitter/…`.
